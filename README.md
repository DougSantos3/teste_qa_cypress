This is a Test QA - project using  [`cypress`](https://www.cypress.io/).

## Getting Started

You need to download the packages:

```bash
node: v22.11.0
```

```bash
npm install
# or
yarn
```

Run tests:

```bash
# Open dashboard cypress
npm run cy:open

# Run tests ui
npm run test:ui

# Run tests api
npm run test:api
```
